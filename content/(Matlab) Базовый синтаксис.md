###### <u>Ввод, вывод</u>
1. Ввод: `a = input('Введите что-нибудь: ');` (`a` $-$ не строка!)
2. Вывод: `disp(a)` (есть ли разница с `display(a)`?)
###### <u>Стандартные функции</u>
1. Математические: `sqrt(), `
2. Массивы: `max(), min(), numel()` (кол-во элементов), `size()` (размер)
###### <u>Логические выражения</u>
1. И: `a && b`
2. ИЛИ: `a || b`
3. НЕ: `~a`

###### <u>Арифметические операции</u>
1. `abd` $-$ модуль
2. `angle` $-$ аргумент комплексного числа
3. `real`, `imag` $-$ действительная и мнимая часть
4. `conj` $-$ комплексное сопряжение
5. `sign` $-$ знак числа
6. `rem(a,b)` $-$ остаток от деления
7. `sqrt` $-$ квадратный корень
8. `exp` $-$ экспоненциальная функция
9. `log` $-$ функция натурального логарифма (`log2`, `log10`)
10. Округление: `ceil`, `fix`, `floor`, `round`
11. Тригонометрические: `sin, cos, tan, cot` (`asin, acos, atan, atan2, acot`)
12. Преобразование систем координат:
	1. `cart2pol` $-$ декартова в полярную / цилиндрическую (`pol2cart`)
	2. `cart2sph` $-$ декартова в сферическую (`sph2cart`)
###### <u>Условия</u>
``` Matlab
if a > 0
	disp('Положительное')
elseif a < 0
	disp('Отрицательное')
else
	disp('Ноль')
end
```
Так же если условия в одну строку:
``` Matlab
if a > 0, disp('Положительное'); else, disp('Отрицательное'); end
```
###### <u>Циклы</u>
Контейнер $-$ либо арифметическая последовательность `n:h:m`, либо матрица. Во втором случае, `i=контейнер(:, столбец)`
```
for i = контейнер
	...
end
```
Функции:
1. `break` $-$ остановить итерации
2. `continue` $-$ переход к следующей итерации
###### <u>Работа с командным окном (Command window)</u>
1. `clc` $-$ очистить все поле в Command window
2. `clear` $-$ очистить переменные
