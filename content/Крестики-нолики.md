Дано поле игры в крестики-нолики. Это корректное поле завершённой игры. То есть либо выиграли крестики, либо нолики, либо игра завершилась вничью. Требуется по виду поля определить результат игры.

**Формат ввода**
Три строки, в каждой по три символа - вид поля. Крестик обозначается 'X' (икс большое), нолик - '0' (ноль), свободная клетка - '.' (точка).

**Формат вывода**
Один символ. Если победили крестики - 'X' (икс большое), если нолики - '0' (ноль), если ничья - '?' (вопросительный знак).

| Ввод                    | Результат |
| ----------------------- | --------- |
| `X.0`<br>`.X0`<br>`..X` | `X`       |
