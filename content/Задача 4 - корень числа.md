Найдите квадратный корень числа $N$ методом Ньютона.

Метод Ньютона заключается в том, что некоторое уравнение типа $f(x)=0$ решается итеративно: есть начальное "грубое" предположение о значении $x,$ потом мы его "уточняем". Обозначим предположение как $x_0,$ а уточняющиеся значения как $x_1,x_2,....$ Поскольку мы хотим найти такое число $x,$ чтобы оно равнялось корню $x=\sqrt{N},$ нехитрыми действиями приводим уравнение к нужному виду $f(x)=x^2-N=0.$

В нашем случае для числа $N\in (0,+\infty)$ начальное положение выберем "наобум": $\sqrt{N}\approx x_0=1.$ Спойлер: из-за простоты функции $\sqrt{N}$ метод всё равно из такой грубой оценки придёт к правильному ответу.

Теперь обсудим, как их предыдущего приближения $x_i$ найти последующее $x_{i+1}.$ Метод Ньютона состоит в формуле: $x_{i+1}=x_i - \frac{f(x_i)}{f'(x_i)}.$ Здесь $f'(x)$ $-$ производная функции, и для $f(x)=x^2-N$ она известна: $f'(x)=2x.$

Собирая всё вместе, получаем $x_{i+1}=x_i - \frac{x^2_i-N}{2x_i}.$

Записав последовательность из 11-ти значений $x_0...x_{10}$ в массив `x` $-$ 1 начальное приближение и 10 шагов метода Ньютона, $-$  отобразите его на графике:
``` Python
import matplotlib.pyplot as plt
plt.plot(x)  # Отображение линии приближений
plt.grid()  # Отображение сетки
plt.show()  # Вывод графика в специальное окно
```
