Найдите квадратный корень числа $N$ методом Ньютона.

Допустим, есть некоторое число $N\in (0,+\infty),$ и начальное предположение: $\sqrt{N}\approx x_0=1.$ Далее, будем уточнять наши предположения $x_1,x_2,...$ методом Ньютона. 

Такой метод находит решение уравнения $f(x)=0.$ В данном случае, $f(x)=x^2-N=0.$ 

Сам метод заключается в уточнении приближений: $x_{i+1}=x_i - \frac{f(x_i)}{f'(x_i)}.$ Здесь $f'(x)$ $-$ производная функции: $f'(x)=2x.$

Собирая всё вместе, получаем $x_{i+1}=x_i - \frac{x^2_i-N}{2x_i}.$

Записав последовательность $x_0...x_{10}$ в массив `x`, отобразите его на графике:
``` Python
import matplotlib.pyplot as plt
plt.plot(x)  # Отображение линии приближений
plt.grid()  # Отображение сетки
plt.show()  # Вывод графика в специальное окно
```
