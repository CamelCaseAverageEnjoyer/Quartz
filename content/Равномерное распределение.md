Пусть есть величина $\xi,$ распределённая равномерно на отрезке $[0,1].$ Эта величина $\xi$ может быть реализована с помощью модуля `random` и записана в переменную $x:$
``` Python
import random
x = random.uniform(0, 1)  # для [0,1] это то же, что и random.random()
```
Для большого числа $N$ (к примеру, 1000), сделайте $N$ реализаций величины $x.$ 
Какая часть из них лежит в  следующих промежутках: $\Delta_1 = [0,0.1], \hskip10px ... \hskip10px \Delta_{10}=[0.9,1]$

Пусть количество реализаций $\xi,$ попавших в промежуток $\Delta_i,$ обозначается $n_i.$ 
Вероятность того, что $\xi$ окажется в $\Delta_i,$ для достаточно больших $N$ записывается формулой: $P_i=P(\xi\in\Delta_i)\approx \frac{n_i}{N}$

Посчитайте массив $p=[P_1,...P_{10}]$ и отобразите его:
``` Python
import matplotlib.pyplot as plt
plt.plot(p)
plt.show()
```
