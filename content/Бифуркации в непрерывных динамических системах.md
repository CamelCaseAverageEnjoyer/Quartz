2. <span style="color:rgb(7, 207, 174)">Точка биффуркации</span> - значение $\mu_0$ в системе, для которого $f(x,\mu_0)$ в компакте $D$ не является грубой
3. <span style="color:rgb(237, 214, 146)">Пример:</span> $\dot{x}=\mu-x^2,\hskip10px x\in\mathbb{R},\mu\in\mathbb{R}, \hskip10px x\in[-\delta , \delta ]$

		1. $\mu<0 \hskip10px \to$ ПР нет
		2. $\mu=0 \hskip10px \to$ ПР: $x=0$ 
		3. $\mu>0 \hskip10px \to$ ПР: $x=\pm\sqrt{\mu}$
	2. Исследуем ПР на гиперболичность
		1. $\mu<0 \hskip10px \to$ -
		2. $\mu=0 \hskip10px \to$ ПР: 1 не гиперболическое
		3. $\mu>0 \hskip10px \to$ ПР: 2 гиперболических

6. <span style="color:rgb(0, 176, 240)">Теорема Сотомайера</span> для седло-узловой биффуркации:
	1. Пусть для системы $f(x,\mu), \hskip10px x\in \mathbb{R}^n, \mu\in\mathbb{R}$   eсть положение равновесия при $\mu_0: \hskip10px f(x_0,\mu_0)=0$
	2. Матрица $A=\frac{\partial f}{\partial x}(x_0,\mu_0)$ имеет простое (кратности 1) собственное число с правым СВ $u$ и левым СВ $w$
	3. Пусть $A$ имеет $k$ собственных значений с $Re(\lambda )<0,$ и $(n-k-1)$ с $Re(\lambda )>0$
	4. Пусть  $\begin{cases} w^T \frac{\partial f}{\partial \mu}(x_0, \mu_0)\neq 0 \\ w^T \frac{\partial^2}{\partial x \partial x}(f(x_0, \mu_0)(u,u)) = w^T \sum_{i,j=1}^n \frac{\partial^2 f(x_0,\mu_0)}{\partial x_i \partial x_j} u_i u_j \neq 0 \end{cases} {}$
	5. Тогда:
		1. $\varepsilon {}$ гладкая кривая из ПР в $\mathbb{R}^n\times \mathbb{R},$  проходящая через $(x_0,\mu_0)$
		2. Нет ПР около $x_0$ при $\mu<\mu_0$ (или $\mu>\mu_0$) и $\exists$ 2 ПР около $x_0$ при $\mu>\mu_0$ (или $\mu<\mu_0$)
		3. 2 ПР около $x_0$ гиперболические, система испытывает седло-узловую биффуркацию
7. <span style="color:rgb(0, 176, 240)">Теорема Сотомайера</span> для транскритической биффуркации:
	1. $\begin{cases} w^T \frac{\partial f}{\partial \mu}(x_0, \mu_0)= 0 \\ w^T \frac{\partial ^2 f}{\partial \mu \partial x}(x_0, \mu_0) u \neq 0 \\ w^T \frac{\partial ^2 f}{\partial x \partial x}(x_0, \mu_0) (u,u) \neq 0 \end{cases} {}$
	2. Тогда слева 2 ПР, справа 2 ПР
8. <span style="color:rgb(0, 176, 240)">Теорема Сотомайера</span> для биффуркации типа "вилка":
	1. $\begin{cases} w^T \frac{\partial f}{\partial \mu}(x_0, \mu_0)= 0 \\ w^T \frac{\partial f}{\partial x}(x_0, \mu_0) u= 0  \\ w^T \frac{\partial ^2 f}{\partial x \partial x}(x_0, \mu_0) (u,u) \neq 0 \\ ... \end{cases} {}$
	2. Тогда слева 2 ПР, справа 2 ПР
9. <u>Развёртка. Универсальная развёртка негрубой ДС.</u>
	1. Пусть есть структурно неустойчивая ДС $\dot{x}=f_0(x)$
		1. То есть $\varepsilon \dot{x}=f(x)+g(x): |g(x)|\leq \varepsilon ,|g'(x)|\leq \varepsilon :$ ФП будет отличаться 
		2. Будем считать ДС точкой: $f_0(x)$ - элемент в пространстве $C^1$-функций на $D$
		3. Можно ли параметризовать все варианты изменения ДС конечным вариантом?
	2. Рассмотрим одномерный случай $\dot{x}=f(x): \hskip10px f(0)=0$
		4. Можем разложить правую часть по Тейлору: $f(x)\to a_k x^k+..., \hskip10px a_k\neq 0, \hskip10px k=\overline{2,\infty  }$
			1. $k\neq 1$, потому что иначе не будет структурной устойчивости (гиперболическая система)
		5. Сделаем замену времени: $d\tau=-a_k dt$
			2. Тогда $\dot{x}'_\tau=-x^k+...=-x^k+\eta g(\eta, x), \hskip10px g$ - гладкая $g\in C^1(D)$
			3. Есть некая константа $\eta$, которая близка к 0
		6. Для такой системы теорема:
	3. <span style="color:rgb(0, 176, 240)">Теорема:</span>
		1. $\forall g\in C'(D)\to \exists$  замена $x=y+\eta h(x), \hskip10px h\in C^1(D)$  и функция $\upsilon (\eta,x):u(0,0)\neq 0,$ что
		2. $u(\eta,x)(-x^k+\eta g(\eta, x))=\mu_1(\eta)+\mu_2(\eta)y+\mu_{k-1}(\eta)y^{k-2} + y^k$
		3. <span style="color:rgb(172, 161, 134)">Замечания:</span>
			1. $x=y+\eta h(x)$ - гомеоморфизм
			2. Значит, все ПР 1-й системы - ПР 2-й ситемы
			3. Производную ДС можно в окрестности привести к некоторому виду, где $(k-1)$ параметров
	4. <span style="color:rgb(7, 207, 174)">Развёртка</span> $f(x,\mu), \hskip2px \mu\in\mathbb{R}^m$ гладкого поля $f_0(x)$ - гладкое поле: $f(x,\mu_0)=f_0(x)$
		1. Здесь $\dot{x}=f_0(x),\hskip10px f_0(x)\in C^1(D)$
	5. <span style="color:rgb(7, 207, 174)">Универсальная развёртка</span> $f^*(x,\mu)$: любая другая развёртка топологически эквивалентна подсемейству из $f^*(x,\mu)$
		2. = Любая система $\dot{x}=f(x)$ топологически эквивалентна $x'_\tau = -x^k+...$, где мы отбрасываем в правой части: $x'_\tau = -x^k$. КоРоЧе, $y'=\mu_1(\eta)+\mu_2(\eta)y+\mu_{k-1}(\eta)y^{k-2} + y^k$ - универсальная развёртка
	6. <span style="color:rgb(237, 214, 146)">Пример:</span>   $\dot{x}=\mu - x^2$   - развёртка структурно неустойчивой ДС  $\dot{x}=-x^2$
