Это ещё один пример обратной задачи.

Для задания выберите себе пару. Придумайте свою хэш-функцию в виде:
``` Python
def my_hash(s):
   a = ... # ваш код
   return a
```
Затем поменяйтесь Python-кодом с партнёром по заданию, попытайтесь "взломать" его функцию, т.е. придумайте обратную функцию. Такая функция по хэшу `a` должна угадывать один ключ `s` или все его возможные варианты.
``` Python
def hack_hash(a):
   s = ... # ваш код
   return s
```

В программировании, хэш-функции $-$ такие функции, которые переводят текст в некоторое число таким образом, что по числу текст практически невозможно восстановить (даже приблизительно!). Хэш функции используются в задачах шифрования.
В Python это реализовано с помощью функции `hash(s)`. Заметьте, что для строк разной длины хэш имеет одинаковую длину.
